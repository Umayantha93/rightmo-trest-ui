# RightMo Test UI (React Frontend)

Modern, responsive frontend for the Product Management System built with React 18 and TypeScript.

## Features

- User authentication (login/logout)
- Protected routes with auth guards
- Responsive product dashboard
- Real-time product search
- Advanced filtering (category, price range)
- Multiple sort options (price, rating, name, date)
- Pagination controls
- Product detail view
- Image upload and update
- Product editing and deletion
- Responsive design (mobile, tablet, desktop)

## Installation & Setup

See the main README.md in the parent directory for complete setup instructions.

## Quick Start

```bash
npm install
npm run dev
```

Application will be available at http://localhost:5173

## Key Routes

- /login - Login page
- /dashboard - Product listing (protected)
- /products/:id - Product detail (protected)

## Technologies

- React 18 with TypeScript
- React Router v6
- Axios for API calls
- Vite for build tooling
- Custom CSS for styling
